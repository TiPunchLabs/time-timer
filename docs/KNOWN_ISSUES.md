## Fragile Zones

### FZ-001: SVG clockwise transform chain
- **Location**: `ClockCircle` component + `src/utils/svg.ts`
- **Description**: The `rotate(90deg) scaleX(-1)` trick is non-obvious. If anyone changes the transform on the arc elements without understanding the chain, the fill direction breaks silently.
- **Mitigation**: Documented in ADR-005, comments in svg.ts

### FZ-002: Timer persistence restore logic
- **Location**: `src/hooks/useTimer.ts` (mount effect)
- **Description**: The restore logic branches on `status` and recalculates elapsed time for running timers. Edge cases: clock changes (DST, manual time adjustment), very long absences (> 4h).
- **Mitigation**: Validated with `isValidPersistedState()`, clamped to 0 minimum

### FZ-003: Circle data calculation edge cases
- **Location**: `src/utils/time.ts` — `getCirclesData()`
- **Description**: The drain order logic (first circle drains first) depends on correct threshold math with `drainedThreshold` and `fullThreshold`. Off-by-one errors here cause visual glitches (wrong circle draining).
- **Mitigation**: Extensive test coverage (162 tests), but edge cases around exact hour boundaries deserve attention
